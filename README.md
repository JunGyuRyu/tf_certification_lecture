# Tensorflow certification을 위한 강의
**쉽고 간단한 모델 개발**

- **시험시간**: **5시간** (**온라인** 시험으로 언제든지 응시가능)
- **유효기간:** **3년**
- **준비물**: **여권사본과 웹캠**(이 지원되는 노트북)
- **IDE**: **Pycharm** 을 활용
- **응시료**: **100$**
- **시험**: 총 **5문제**이며 **기본적인 회귀, 이미지 분류(CNN 등),** **텍스트 분석** 등으로 구성
- 내 PC에 **GPU가 없다면**: **코랩과 PyCharm을 동시에 사용**
- **강의**는 무료 기간인 **5일안에 끝내기**
