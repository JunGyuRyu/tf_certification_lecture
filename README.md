# Tensorflow certification을 위한 강의
**쉽고 간단한 모델 개발**
---
- **응시료:** 100$
- **유효기간:** 3년
- **시험시간:** 5시간 (온라인, 언제든지 응시가능)
- **준비물:** 여권사본과 웹캠
- **IDE:** Pycharm
- **시험:** 총 5문제, 기본적인 회귀, 이미지 분류(CNN 등), 텍스트 분석 등으로 구성
- **GPU가 없다면**: 코랩과 PyCharm을 동시 사용
